# Roadmap

## Proximas evolucoes recomendadas

1. Banco local criptografado no mobile para anexos e historico maior.
2. Shells nativos Flutter completos com bloqueio de preview no app switcher por plataforma.
3. Suporte multilocale com conteudo por pais/regiao.
4. Painel administrativo para artigos educativos.
5. Exportacao segura de relato em PDF criptografado.
6. Modo voz com pipeline moderado e seguro.
7. Busca opcional de servicos de apoio por geolocalizacao.
8. Observabilidade privacidade-first com eventos anonimizados.
9. Rate limit distribuido via Redis no backend.
10. Criptografia de campos sensiveis no PostgreSQL com KMS/segredo dedicado.
